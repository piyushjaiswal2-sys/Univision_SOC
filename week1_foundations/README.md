# Week 1 — Computing Without Fear

**Focus:** Scratch, Blockly, flowcharts, and the core logic primitives behind every system.

The goal of Week 1 is to see a real-time AI vision system as what it actually is: *a pipeline of small logical decisions* — capture a frame, clean it, analyze it, validate it, report it, store it, explain it. No neural networks yet.

## Contents
- [`concepts.md`](concepts.md) — the homework write-up: variables, loops, events, and conditions explained in one page.
- [`camera_alert_flowchart.md`](camera_alert_flowchart.md) — the lab deliverable: a flowchart for a motion-triggered camera alert.

## Lab

- Draw a camera-alert flowchart (see `camera_alert_flowchart.md`).
